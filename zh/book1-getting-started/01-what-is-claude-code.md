# 第一章：什么是 Claude Code？

## 想象一下这个场景

凌晨十一点，你盯着一堆看不懂的代码，不知道从哪里下手。你打开微信，发消息给你那个学计算机的朋友："哥们，帮我看看这个程序怎么回事？"

你朋友回复了，给了你一些建议。但问题是——他只看到你复制粘贴过去的那几行代码。他没法直接在你电脑上操作，没法帮你跑一下程序看看报什么错，也没法帮你打开其他文件对照着看。他只能靠你的描述，给你出主意。

现在，想象另一种情况：你那位朋友就坐在你旁边，他可以直接看你的屏幕，可以亲手翻阅你项目里的每一个文件，可以帮你在电脑上运行命令，甚至可以直接帮你修改代码——所有这一切，都不需要你去复制粘贴，不需要你去描述"大概长这样"。

**这就是 Claude Code 的感觉。**

---

## Claude Code 是什么？

Claude Code 是 Anthropic（Claude AI 的开发公司）推出的一款 **AI 编程工具**，它运行在你电脑的 **终端（terminal）** 里。

如果你不熟悉"终端"这个词：终端就是那个黑色或白色的窗口，里面只有文字，没有漂亮的按钮和图标。程序员们用它来运行命令、管理文件、启动程序。Claude Code 就住在这个地方。

> **类比时间：** 如果 ChatGPT 或者网页版 Claude 是"给朋友发微信"，那 Claude Code 就是"让那位朋友直接搬来你家，坐在你电脑旁边帮你干活"。

一句话定义：**Claude Code 是一个能读你的代码、写代码、跑命令、帮你管理整个项目的 AI 助手，它直接在你的电脑上工作。**

---

## Claude Code 能做什么？

当你把 Claude Code 启动在某个项目文件夹里，它能做的事情远超你的想象：

### 读懂你的代码
不管你的项目有多少文件，用什么编程语言写的，Claude Code 都能翻阅、理解，然后告诉你"这个项目是干什么的"、"这里有什么问题"、"这个功能在哪个文件里"。

### 写代码和修改文件
你说"帮我加一个登录页面"，它不只是告诉你怎么写——它会直接帮你**创建文件、写好代码**，然后问你"这样行吗？"

### 跑命令
想运行测试？启动开发服务器？安装依赖？Claude Code 可以直接在你的终端里执行这些命令，就像一个真实的程序员坐在那里敲键盘一样。

### 搜索网页
遇到不认识的错误信息？Claude Code 可以直接上网搜索文档、查找解决方案，不需要你切换到浏览器再复制粘贴。

### 管理 Git 版本控制
"帮我提交这次改动"、"创建一个新分支"、"看看我改了哪些文件"——这些 Git 操作都可以用自然语言完成。

---

## 和网页版 Claude / ChatGPT 有什么不同？

这个问题很关键，很多人一开始搞混了。

| 特性 | 网页版 Claude / ChatGPT | Claude Code |
|------|------------------------|-------------|
| 访问你的文件 | 你手动粘贴进去 | 直接读取你电脑上的文件 |
| 修改你的代码 | 只能给你代码片段 | 直接帮你改文件 |
| 运行命令 | 不能 | 可以（有你的许可） |
| 搜索网页 | 部分支持 | 支持 |
| 理解整个项目 | 有限（靠你粘贴） | 可以扫描所有文件 |
| 记住上下文 | 一个对话窗口 | 当前会话内记住所有操作 |

换一个生活化的比方：

- **网页版 Claude** 像是**远程顾问**：你把问题描述给他听，他给你建议，但所有动手的活还是你来。
- **Claude Code** 像是**上门服务的工程师**：他来到你家（你的电脑），直接看你的实际情况，动手帮你解决问题。

---

## Claude Code 不只是"给程序员用的"

这是很多人的误解。是的，Claude Code 的名字里有"Code"，但它能帮你做的事情远不止写程序：

- 帮你整理一堆文本文件
- 自动化重复性的文件操作
- 搜索和整理信息
- 生成报告、文档、README
- 管理项目文件结构

只要是能在电脑上用命令行完成的事，Claude Code 都能参与。

## 这本书是写给谁的？

这本书写给**所有好奇 Claude Code 的人**，包括：

- **完全没有编程经验的人** — 你不需要先学编程才能用 Claude Code。本书会一步步带你从零开始。
- **有一点编程基础的人** — 你已经能写一点点代码，但想用 AI 把效率提高 10 倍。
- **想了解 AI 工具的人** — 你对 AI 助手感兴趣，想知道"在终端里的 AI"到底是什么体验。

本书**不假设你**：
- 懂 Linux 命令
- 会用 Git
- 懂任何编程语言

每一个概念我们都会从头解释，遇到技术词汇我们都会打比方。你需要的只是一台电脑和一点点好奇心。

---

## 在开始之前，记住一件事

Claude Code 是一个工具，不是魔法。它很强大，但它也会犯错。它做的所有重要操作都会先征求你的同意，而你永远是最后的决策者。

把它想成一个**非常能干但需要你把关的助手**——而不是一个你可以完全托付、完全放手的系统。
